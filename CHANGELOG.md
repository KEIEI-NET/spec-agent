## 移行ガイド

### v1.x から v2.0 への移行

v2.0.0 は後方互換性を保っているため、特別な移行作業は不要です。
ただし、以下の改善を活用することを推奨します：

1. **新しいインストーラーの使用**
   ```bash
   python install.py
   ```

2. **新しいドキュメントの確認**
   - USAGE.md で新機能を確認
   - API_REFERENCE.md でAPI仕様を確認

3. **設定ファイルの更新（オプション）**
   ```json
   {
     "version": "2.0.0",
     "features": {
       "auto_validation": true,
       "enhanced_error_handling": true
     }
   }
   ```
